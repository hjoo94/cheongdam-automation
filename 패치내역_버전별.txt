- 스레드 자영업 글 초안 기능 추가
  - 최근 7일 기준 자영업 키워드 흐름을 참고해 스레드 글 초안 생성
  - Threads API 토큰이 없거나 권한 오류가 나도 직접 붙여넣은 참고 글로 초안 생성 가능
  - 원문 복사 방지를 위해 참고 글의 분위기와 문제의식만 사용하고 새 문장으로 작성
  - 초보자용 사용법 문서 추가: 스레드_자영업_초안_사용법.txt

- 쿠팡이츠 블라인드 해피톡 클릭 검증 강화
  - 실제 클릭 가능한 버튼/선택지 또는 포인터 요소만 클릭 성공 후보로 인정
  - 단계 클릭 후 화면 변화가 없으면 클릭 완료로 처리하지 않도록 수정
  - 단순 텍스트를 클릭 성공으로 오판하던 문제 방지

- 배포
  - 고객 앱: Cheongdam Bot Setup 1.0.15.exe
  - 서버 패치: chungdam-license-server-update-1.0.15.tar.gz

v1.0.23
============================================================

- 쿠팡이츠 블라인드 최종 안정화
  - 해피톡이 이미 서비스 선택 화면에 들어와 있으면 "신규 상담 시작하기" 단계를 건너뛰도록 수정
  - 각 해피톡 단계 클릭 후 다음 단계 문구가 실제로 감지되어야만 다음 단계로 진행하도록 보강
  - 클릭은 됐지만 화면이 다음 단계로 넘어가지 않은 경우 성공 처리하지 않고 실패 로그/캡처를 남기도록 수정

- 쿠팡 다음 페이지 안정화
  - 깨진 페이지네이션 한글 정규식과 로그 문구 복구
  - 삭제/제거/닫기/취소 계열 버튼을 다음 페이지 후보에서 명확히 제외
  - 현재 페이지 그룹의 마지막 페이지에서 다음 후보가 없으면 반복 디버그 덤프 대신 정상 종료 로그를 남기도록 수정

- 배포
  - 고객 앱: Cheongdam Bot Setup 1.0.23.exe

v1.0.24
============================================================

- 네이버 메일과 배민 블라인드 로그 매칭 안정화
  - 네이버 메일 본문의 수신일을 리뷰 작성일로 오인하지 않도록 수정
  - 네이버 메일 화면 전체 텍스트가 리뷰 본문으로 들어오면 매칭 후보에서 제외
  - 신청서 폼에서 리뷰번호, 리뷰 작성일, 게시물 내용을 직접 추출해 배민 블라인드 로그와 매칭하도록 보강
  - 리뷰번호/본문 매칭을 우선 사용하고, 작성일만 있는 경우에는 해당 날짜 로그가 1건일 때만 제한적으로 매칭

- 배포
  - 고객 앱: Cheongdam Bot Setup 1.0.24.exe
