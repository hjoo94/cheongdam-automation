스레드 자영업 글 초안

모자이크 캡처 (스레드에 사진으로 첨부)

아래 미리보기는 화면 표시용입니다. 실제 업로드는 저장 폴더의 PNG 파일을 사용하세요. 모자이크는 자동 처리이며, 민감 정보는 반드시 직접 한 번 더 확인하세요.

이미지를 선택한 뒤 초안을 생성하면 여기에 미리보기가 나타납니다.
공식 API 권한이 없거나 검색을 끈 경우에도 리뷰·답글 자료로 초안을 생성합니다. 원문 문장과 고유 사례는 복사하지 않습니다.
