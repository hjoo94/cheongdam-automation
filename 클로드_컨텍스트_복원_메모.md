## ▶ 긴급·운영 시 참고

  **Git 충돌**     → 가능하면 merge/rebase. **`git push --force`는 팀 합의·비상 시에만** 검토.  
  **서버 장애**    → Lightsail 콘솔에서 인스턴스 재시작, **4300 포트·프로세스(pm2 등)** 확인.  
  **버전 불일치**  → 서버 `.env`의 `CLIENT_APP_*` → 없으면 `server.js` 폴백. **`GET /api/updates/client/latest` JSON**으로 최종 확인.  
  **IP 불일치**    → `PUBLIC_BASE_URL` + 클라이언트 `serverBaseUrl` **동시** 점검.  
  **Git 푸시 실패** → **`git add .`** 에 **대용량 ZIP/설치파일**이 섞이지 않았는지 확인 (GitHub 100MB 제한).
